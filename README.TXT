3. Build Instructions
=====================
To run the Backend Java application locally:
cd openhome-backend
./mvnw  package && java -jar target/openhome-0.0.1-SNAPSHOT.jar

To run the frontend reactJS application:
cd openhome-frontend
npm install && npm start
